Interview exprience at Orcale..



There was 1 Online Assessment Test, followed by 2 online techinal questions and 1 offline question.

Online Assessment Test

So, the Online Assessment test was also offline at our instutite ,This round had one coding problem, one SQL problem, one API implementation problem and 6 MCQ's. Coding problem was a LC medium, SQL one was easy, and API implementation was fairly basic.

1st online Interview

So, around more than 50+ students cleared OA round. This was a short interview started with basic introduction and then on hackerrank Peer-to-Peer coding platform they asked 1 DSA question ( largest-number ) and some very basic OOPs and CS fundament questions. Right after this round in around 10 mins my next interview was scheduled.

2nd online Interview

Before this round started i was not able to find a room to sit and give the interview, so i told the interviewer to give me some time to arrange a room at instutite. The interview started with bsic introduction and in my introduction I mentioned about Table Tennis, we discuss on TT open tournments. After that again on Peer-to-Peer hackerrank platform started with a 3 DSA question All Combination that adds upto given number then a question on Binary Search and some variation of the same and lastly some OOPs Question.

Last Offline Interview

This was the final round and a senior Developer from Oracle Ghandinager came to take interview. Again started with basic introdution , then he asked me if I knew about the game WORDLE , I didnt knew the rules So he told me to took out my phone and play the game after playing the game for 5-10 mins we discussed about the game rules, after that he told me implement it on pen-Paper. I wrote a CLI program in C++ because of time constraints. (I was thinking of making a flutter application xD).

Verdict

Selected as Applications Developer (Full Time).
